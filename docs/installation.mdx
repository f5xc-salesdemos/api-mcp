## MCPB Bundle (Claude Desktop)

1. Download `.mcpb` from [GitHub Releases](https://github.com/f5xc-salesdemos/api-mcp/releases)
2. Double-click or drag into Claude Desktop
3. Click **Install**

## npx (Recommended)

```bash
npx @f5xc-salesdemos/api-mcp
```

## npm Global

```bash
npm install -g @f5xc-salesdemos/api-mcp
f5xc-api-mcp
```

## Docker

```bash
docker pull ghcr.io/f5xc-salesdemos/api-mcp:latest
docker run -it ghcr.io/f5xc-salesdemos/api-mcp
```

## From Source

```bash
git clone https://github.com/f5xc-salesdemos/api-mcp.git
cd api-mcp
npm install && npm run build && npm start
```

## Verify Installation

### Version Check

```bash
npx @f5xc-salesdemos/api-mcp --version
```

### MCP Protocol Test

```bash
echo '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"capabilities":{}}}' | npx @f5xc-salesdemos/api-mcp
```
