Cache Purge provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-cdn-cache-purge-create` | Purge CDN Cache. |

## Parameters

| Parameter | Description | Example |
