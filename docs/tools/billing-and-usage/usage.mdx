Usage provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-billingandusage-usage-list` | Get Quota Usage. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace The tenant this item belongs to. | `system` |

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/web/namespaces/$${PARAM}/quota/usage" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
