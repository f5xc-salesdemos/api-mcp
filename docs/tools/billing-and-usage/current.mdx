Current provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-billingandusage-current-list` | Get current usage plan. |

```bash
# List resources

  -H "Authorization: APIToken ${F5XC_API_TOKEN}"
```
