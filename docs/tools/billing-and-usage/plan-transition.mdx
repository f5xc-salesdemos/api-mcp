Plan Transition provides tools for creating, and listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-billingandusage-plan-transition-create` | InitiatePlanTransition. |
| `f5xc-api-billingandusage-plan-transition-list` | GetPlanTransition. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace This namespace is not used, all requests are stored under system namespace. | `system` |

### Query Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `id` | ID of the plan transition request. | `-` |


## Example Usage

Ask Claude to help you work with Plan Transition resources:

### Create Plan Transition

> "Create a plan-transition named 'example' in the 'production' namespace"

### List Plan Transitions

> "List all plan-transitions in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/no_auth/namespaces/system/billing/plan_transition" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/web/namespaces/$${PARAM}/billing/plan_transition" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @plan_transition.json

```
