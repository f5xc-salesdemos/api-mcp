Swap Primary provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-billingandusage-swap-primary-create` | Make payment method secondary. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `name` | Name The name of the payment method object to be updated. | `my-resource` |
| `namespace` | Namespace The namespace in which the payment method object is present. | `system` |


## Example Usage

Ask Claude to help you work with Swap Primary resources:

### Create Swap Primary

> "Create a swap-primary named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/web/namespaces/$${PARAM}/billing/payment_method/$${PARAM}/swap-primary" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @swap_primary.json

```
