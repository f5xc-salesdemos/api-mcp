## CURL Examples

```bash
# List resources

  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json"

# Get specific resource
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/config/namespaces/${NAMESPACE}/segments/<name>" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json"

# Create resource

  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \

  -d @segment.json

# Delete resource

  -H "Authorization: APIToken ${F5XC_API_TOKEN}"
```
