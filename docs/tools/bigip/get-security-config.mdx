## Tools

| Tool | Description |

## Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace x-required... | `system` |

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/config/namespaces/$${PARAM}/bigip_virtual_servers/get_security_config" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @get_security_config.json

```
