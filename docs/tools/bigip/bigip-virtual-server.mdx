Bigip Virtual Server provides tools for listing, retrieving, and updating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-bigip-bigip-virtual-server-get` | Get BIG-IP virtual server. |
| `f5xc-api-bigip-bigip-virtual-server-list` | List BIG-IP virtual server. |
| `f5xc-api-bigip-bigip-virtual-server-update` | Update BIG-IP virtual server. |
