Register provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-marketplace-register-create` | Register New AWS Account. |

## Example Usage

Ask Claude to help you work with Register resources:

### Create Register

> "Create a register named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/no_auth/namespaces/system/aws/f5xc-saas/register" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @register.json

```
