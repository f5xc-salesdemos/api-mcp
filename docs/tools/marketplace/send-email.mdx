Send Email provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-marketplace-send-email-create` | Send Signup Email. |

## Example Usage

Ask Claude to help you work with Send Email resources:

### Create Send Email

> "Create a send-email named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/no_auth/namespaces/system/f5xc-saas/signup/send_email" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @send_email.json

```
