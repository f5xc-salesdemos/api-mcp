Activation Statu provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-marketplace-activation-statu-list` | Addon Service Activation Status. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `addon_service` | Addon_service Name of the addon_service. | `-` |
