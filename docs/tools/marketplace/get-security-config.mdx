## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-marketplace-get-security-config-create` | Get Security Config for Third Party Application. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace x-required... | `system` |

| Option | Description | Recommended |
|--------|-------------|-------------|

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/config/namespaces/$${PARAM}/third_party_application/get_security_config" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @get_security_config.json

```
