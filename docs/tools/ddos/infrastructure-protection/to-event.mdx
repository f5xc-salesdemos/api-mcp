To Event provides tools for updating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-ddos-to-event-update` | Link Alert to Event. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `alert_id` | Alert ID Alert to be link with an event. | `-` |
| `namespace` | Namespace This request is supported only in system namespace. | `system` |
