Acces provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-ddos-acces-list` | Customer access. |

## Example Usage

Ask Claude to help you work with Acces resources:

### List Access

> "List all access in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/infraprotect/namespaces/system/infraprotect/access" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
