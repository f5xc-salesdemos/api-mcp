API Endpoint Protection provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-api-api-endpoint-protection-create` | Suggest API endpoint protection rule. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace x-required... | `system` |


## Example Usage

Ask Claude to help you work with API Endpoint Protection resources:

### Create API Endpoint Protection

> "Create a api-endpoint-protection named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/config/namespaces/$${PARAM}/api_sec/rule_suggestion/api_endpoint_protection" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @api_endpoint_protection.json

```
