Move To Inventory provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-api-move-to-inventory-create` | Move To API Inventory. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `name` | Name The name of the API Definition for the current request. | `my-resource` |
| `namespace` | Namespace The namespace of the API Definition for the current request. | `system` |


## Example Usage

### Create Move To Inventory

> "Create a move-to-inventory named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/config/namespaces/$${PARAM}/api_definitions/$${PARAM}/move_to_inventory" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @move_to_inventory.json

```
