Evaluate provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-api-evaluate-create` | Evaluate API Group. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Namespace of the API Group for the current request. | `system` |


## Configuration Choices

This resource includes mutually exclusive configuration options:

### scope_choice

| Option | Description | Recommended |
|--------|-------------|-------------|
| `api_group.bigip_virtual_server` | Bigip virtual server |  |
| `api_group.cdn_loadbalancer` | Cdn loadbalancer |  |
| `api_group.http_loadbalancer` | Http loadbalancer |  |


## Example Usage

Ask Claude to help you work with Evaluate resources:

### Create Evaluate

> "Create a evaluate named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/ml/data/namespaces/$${PARAM}/app_api_groups/evaluate" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @evaluate.json

```
