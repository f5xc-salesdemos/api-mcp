Data Exposure provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-api-data-exposure-create` | Suggest sensitive data rule. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace x-required... | `system` |


## Example Usage

Ask Claude to help you work with Data Exposure resources:

### Create Data Exposure

> "Create a data-exposure named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/config/namespaces/$${PARAM}/api_sec/rule_suggestion/data_exposure" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @data_exposure.json

```
