Show routes exported / imported via BGP.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-network-bgp-route-list` | Show BGP Routes. |

## Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|

| `site` | Site Name | `-` |
