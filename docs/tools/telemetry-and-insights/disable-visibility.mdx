Disable Visibility provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-telemetryandinsights-disable-visibility-create` | Disable visibility in all workspaces. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `name` | Service Name x-required... | `example-resource` |
| `namespace` | Namespace x-required... | `system` |

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/discovery/namespaces/$${PARAM}/discovered_services/$${PARAM}/disable_visibility" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @disable_visibility.json

```
