Refresh Token provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-aiservices-refresh-token-create` | Refresh Token. |

## Example Usage

Ask Claude to help you work with Refresh Token resources:

### Create Refresh Token

> "Create a refresh-token named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/ai_data/bfdp/namespaces/system/bfdp/refresh_token" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @refresh_token.json

```
