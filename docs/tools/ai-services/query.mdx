Query provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-aiservices-query-create` | AI Assistant Query. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Namespace of the HTTP Load Balancer for current request. | `system` |


## Example Usage

Ask Claude to help you work with Query resources:

### Create Query

> "Create a query named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/gen-ai/namespaces/$${PARAM}/query" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @query.json

```
