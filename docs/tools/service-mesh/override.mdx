Override provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-servicemesh-override-list` | Get Override. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `app_type_name` | App Type App Type for current request. | `-` |
| `namespace` | Namespace Namespace of the App type for current request. | `system` |


## Example Usage

Ask Claude to help you work with Override resources:

### List Overrides

> "List all overrides in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/ml/data/namespaces/$${PARAM}/app_types/${PARAM}/overrides" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
