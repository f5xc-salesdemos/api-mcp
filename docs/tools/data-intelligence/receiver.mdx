Receiver provides tools for creating, listing, retrieving, updating, and deleting resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-dataintelligence-receiver-create` | Create Data Delivery. |
| `f5xc-api-dataintelligence-receiver-get` | Get Data Delivery. |
| `f5xc-api-dataintelligence-receiver-list` | List Data Delivery. |
| `f5xc-api-dataintelligence-receiver-update` | Update Data Delivery. |
| `f5xc-api-dataintelligence-receiver-delete` | Delete Data Delivery. |

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/data-intelligence/namespaces/$${PARAM}/receivers" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

# Get specific resource
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/data-intelligence/namespaces/$${PARAM}/receivers/$${PARAM}" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/data-intelligence/namespaces/$${PARAM}/receivers" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @receiver.json

# Delete resource
curl -X DELETE "https://${TENANT}.console.ves.volterra.io/api/data-intelligence/namespaces/$${PARAM}/receivers/$${PARAM}" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"
```
