DataSet provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-dataintelligence-dataset-list` | Get Data Sets. |

## Example Usage

Ask Claude to help you work with DataSet resources:

### List DataSets

> "List all dataSets in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/data-intelligence/namespaces/system/dataSets" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
