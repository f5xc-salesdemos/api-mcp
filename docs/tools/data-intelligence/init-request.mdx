Init Request provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-dataintelligence-init-request-create` | Enable Data Intelligence. |

## Example Usage

Ask Claude to help you work with Init Request resources:

### Create Init Request

> "Create a init-request named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/data-intelligence/namespaces/system/init-request" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @init_request.json

```
