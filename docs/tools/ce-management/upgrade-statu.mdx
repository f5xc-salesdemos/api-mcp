Upgrade Statu provides tools for retrieving resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-cemanagement-upgrade-statu-get` | Get Upgrade Status. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `name` | Name Fetch upgrade status for the name of site. | `example-resource` |
| `namespace` | Namespace Fetch upgrade status for the given namespace. | `system` |
