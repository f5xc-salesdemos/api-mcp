Stop Tcpdump provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-support-stop-tcpdump-create` | Stop Tcpdump. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Namespace for which the request is sent (system) | `system` |
| `site` | Site Name Name of the site on which Tcpdump was run. | `-` |


## Example Usage

Ask Claude to help you work with Stop Tcpdump resources:

### Create Stop Tcpdump

> "Create a stop-tcpdump named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/operate/namespaces/$${PARAM}/sites/${PARAM}/ver/stop_tcpdump" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @stop_tcpdump.json

```
