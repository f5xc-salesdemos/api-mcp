## Tools

| `f5xc-api-support-tax-exempt-request-create` | Tax exemption verification request. |

## Example Usage

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/web/namespaces/system/customer_support/tax_exempt_request" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @tax_exempt_request.json

```
