Dhcp Lease provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-support-dhcp-lease-list` | Show DHCP Leases. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Namespace for which the request is sent (system) | `system` |
| `site` | Site Name Name of the site. | `-` |


## Example Usage

Ask Claude to help you work with Dhcp Lease resources:

### List Dhcp Leases

> "List all dhcp-leases in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/operate/namespaces/$${PARAM}/sites/${PARAM}/ver/dhcp_leases" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
