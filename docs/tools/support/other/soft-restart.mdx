Soft Restart provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-support-soft-restart-create` | Soft restart. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Namespace for which the request is sent (system) | `system` |
| `node` | Node Name Name of the node. | `-` |
| `service` | Service Name Name of the service. | `-` |
| `site` | Site Name Name of the site. | `-` |


## Example Usage

Ask Claude to help you work with Soft Restart resources:

### Create Soft Restart

> "Create a soft-restart named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/operate/namespaces/$${PARAM}/sites/${PARAM}/vpm/debug/${PARAM}/${PARAM}/soft-restart" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @soft_restart.json

```
