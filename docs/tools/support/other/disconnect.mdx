Disconnect provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-support-disconnect-create` | Disconnect. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Namespace for which the request is sent (system) | `system` |
| `node` | Node Name Name of the node. | `-` |
| `site` | Site Name Name of the site. | `-` |


## Example Usage

Ask Claude to help you work with Disconnect resources:

### Create Disconnect

> "Create a disconnect named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/operate/namespaces/$${PARAM}/sites/${PARAM}/vpm/lte/${PARAM}/disconnect" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @disconnect.json

```
