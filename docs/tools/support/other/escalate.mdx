Escalate provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-support-escalate-create` | Escalate a ticket. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `name` | Name The name of the customer support ticket object to be escalated. | `my-resource` |
| `namespace` | Namespace The namespace in which the support ticket object is present in. | `system` |


## Example Usage

Ask Claude to help you work with Escalate resources:

### Create Escalate

> "Create a escalate named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/web/namespaces/$${PARAM}/customer_support/$${PARAM}/escalate" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @escalate.json

```
