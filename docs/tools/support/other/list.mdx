List provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-support-list-list` | List USB devices. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Namespace for which the request is sent (system) | `system` |
| `node` | Node Name Name of the node. | `-` |
| `site` | Site Name Name of the site. | `-` |


## Example Usage

Ask Claude to help you work with List resources:

### List Lists

> "List all lists in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/operate/namespaces/$${PARAM}/sites/${PARAM}/vpm/usb/${PARAM}/list" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
