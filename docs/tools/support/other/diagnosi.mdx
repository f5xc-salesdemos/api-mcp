Diagnosi provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-support-diagnosi-list` | Diagnosis. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `site` | Site Name Name of the site. | `-` |

### Query Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `console_user` | Name of the console user who runs this command. | `-` |


## Example Usage

Ask Claude to help you work with Diagnosi resources:

### List Diagnosis

> "List all diagnosis in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/operate/namespaces/system/sites/${PARAM}/vpm/debug/global/diagnosis" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
