DNS Monitor Summary provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-observability-dns-monitor-summary-list` | Get DNS Monitor Summary. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Request namespace. | `system` |

### Query Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `end_time` | End_time. End time for requested monitor data (default: current time) | `-` |
| `monitor_name` | Monitor_name. Monitor name. | `-` |
| `start_time` | Start_time. X-required Start time for requested monitor data. | `-` |


## Example Usage

Ask Claude to help you work with DNS Monitor Summary resources:

### List DNS Monitor Summarys

> "List all dns-monitor-summarys in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/observability/synthetic_monitor/namespaces/$${PARAM}/dns-monitor-summary" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
