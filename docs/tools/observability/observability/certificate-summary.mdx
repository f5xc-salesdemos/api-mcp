Certificate Summary provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-observability-certificate-summary-list` | Get Certificate Summary. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Request namespace. | `system` |

### Query Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `period_in_days` | Period_in_days. Maximum number of days until certificate expiration. | `-` |


## Example Usage

Ask Claude to help you work with Certificate Summary resources:

### List Certificate Summarys

> "List all certificate-summarys in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/observability/synthetic_monitor/namespaces/$${PARAM}/certificate-summary" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
