Global Summary provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-observability-global-summary-list` | Get Global Summary. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Request namespace. | `system` |

### Query Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `monitor_type` | Monitor_type. Monitor type. | `-` |


## Example Usage

### List Global Summarys

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/observability/synthetic_monitor/namespaces/$${PARAM}/global-summary" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
