Monitor History provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-observability-monitor-history-list` | Get Monitor History. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Request namespace. | `system` |

### Query Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `end_time` | End_time. End time for requested history (default: current time) | `-` |
| `monitor_name` | Monitor_name. X-required Monitor name. | `-` |
| `monitor_type` | Monitor_type. X-required Monitor type. | `-` |
| `start_time` | Start_time. X-required Start time for requested history. | `-` |
| `step_size` | Step_size. Step size for time series aggregation. | `-` |
