HTTP Monitor Summary provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-observability-http-monitor-summary-list` | Get HTTP Monitor Summary. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Request namespace. | `system` |

### Query Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `end_time` | End_time. End time for requested monitor data (default: current time) | `-` |
| `monitor_name` | Monitor_name. X-required Monitor name. | `-` |
| `start_time` | Start_time. X-required Start time for requested monitor data. | `-` |

```bash
# List resources

  -H "Authorization: APIToken ${F5XC_API_TOKEN}"
```
