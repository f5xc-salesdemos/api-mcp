Returns TLS summary of all HTTPS monitors running in namespace.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-observability-tls-summary-list` | GET TLS Summary. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace | `-` |
