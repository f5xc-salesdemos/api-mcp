TLS Summary provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-observability-tls-summary-list` | Get TLS Summary. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Request namespace. | `system` |


## Example Usage

Ask Claude to help you work with TLS Summary resources:

### List TLS Summarys

> "List all tls-summarys in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/observability/synthetic_monitor/namespaces/$${PARAM}/tls-summary" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
