Global History provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-observability-global-history-list` | Get Global History. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Request namespace. | `system` |

### Query Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `end_time` | End_time. End time for requested history (default: current time) | `-` |
| `monitor_type` | Monitor_type. Monitor type. | `-` |
| `start_time` | Start_time. Start time for requested history. | `-` |
| `step_size` | Step_size. Step size for time series aggregation. | `-` |


## Example Usage

Ask Claude to help you work with Global History resources:

### List Global Historys

> "List all global-historys in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/observability/synthetic_monitor/namespaces/$${PARAM}/global-history" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
