History provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-observability-history-list` | Get Alerts History. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace namespace to scope the listing of alerts. | `system` |

### Query Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `end_time` | Fetch alerts whose timestamp &lt;= end_time format: unix_timestamp\|RFC 3339... | `-` |
| `filter` | HighDiskUsage\", severity=\"critical\"\}" List of matchers to filter alert by. | `-` |
| `start_time` | Fetch alerts whose timestamp >= start_time format: unix_timestamp\|RFC 3339... | `-` |
