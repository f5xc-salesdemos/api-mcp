Firewall Log provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-observability-firewall-log-create` | Firewall Logs Query. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace fetch Firewall logs for a given namespace. | `system` |


## Example Usage

Ask Claude to help you work with Firewall Log resources:

### Create Firewall Log

> "Create a firewall-log named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/data/namespaces/$${PARAM}/firewall_logs" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @firewall_log.json

```
