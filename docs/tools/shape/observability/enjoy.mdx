Enjoy provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-shape-enjoy-create` | Enjoy Dashboard. |

## Example Usage

### Create Enjoy

> "Create a enjoy named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/shape/recognize/namespaces/system/recognize/addon/dashboard/enjoy" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @enjoy.json

```
