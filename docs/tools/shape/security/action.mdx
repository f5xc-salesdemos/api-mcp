| Tool | Description |
|------|-------------|
| `f5xc-api-shape-action-create` | Malicious Traffic Overview in actions. |

## Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace... | `system` |


## Example Usage

Ask Claude to help you work with Action resources:

### Create Action

> "Create a action named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/shape/bot/namespaces/$${PARAM}/reporting/traffic/malicious/overview/actions" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @action.json

```
