Ua provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-shape-ua-create` | Malicious Report Transactions UA. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace... | `system` |


## Example Usage

Ask Claude to help you work with Ua resources:

### Create Ua

> "Create a ua named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/shape/bot/namespaces/$${PARAM}/reporting/malicious/transactions/ua" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @ua.json

```
