## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|

| `version` | Version Version of the stored_object in "v\{n\}-\{YY\}-\{MM\}-\{DD\}" formatted string, where n is version number and YY/MM/DD is year, month and date. | `-` |
