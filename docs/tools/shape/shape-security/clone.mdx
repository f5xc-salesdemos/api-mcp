Clone provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-shape-clone-create` | Clone Alert Template. |

## Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/shape/alerts/namespaces/$${PARAM}/alert_templates/$${PARAM}/clone" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @clone.json

```
