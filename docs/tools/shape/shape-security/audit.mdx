Audit provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-shape-audit-list` | Get SAFE Block table list. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace... | `system` |

### Query Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `dimt` | Account_id/device_id how to apply the blocking rule. | `-` |
| `dimv` | The relevant account_id/device_id. | `-` |
| `version` | The API version to use. | `-` |
