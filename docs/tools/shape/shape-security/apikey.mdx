## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-shape-apikey-list` | API Key. |

# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/shape/bot/namespaces/system/apikey" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"
