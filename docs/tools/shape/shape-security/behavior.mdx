Behavior provides tools for retrieving resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-shape-behavior-get` | List Behaviors By Script. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `id` | ID Script ID. | `-` |
| `namespace` | Namespace Namespace. | `system` |

### Query Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `end_time` | X-required format: unix epoch timestamp in seconds... | `-` |
| `start_time` | X-required format: unix epoch timestamp in seconds... | `-` |
| `type` | X-required Filter based on type of behaviors... | `-` |
