Template provides tools for retrieving resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-shape-template-get` | Template Connector. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `name` | Application Name Protected application name. | `example-resource` |
| `namespace` | Namespace namespace is used to scope the query. | `system` |

```bash
# Get specific resource
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/shape/bot/namespaces/$${PARAM}/protected_applications/$${PARAM}/template" \
```
