## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-shape-version-get` | Bot Allowlist Policy Versions. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `name` | Policy name Policy name. | `example-resource` |
| `namespace` | Namespace namespace is used to scope the query. | `system` |
