Config provides tools for retrieving resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-shape-config-get` | Connector Configuration. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `name` | Application Name Protected application name. | `my-resource` |
| `namespace` | Namespace namespace is used to scope the query. | `system` |

## CURL Examples

```bash
# Get specific resource
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/shape/bot/namespaces/$${PARAM}/protected_applications/$${PARAM}/config" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
