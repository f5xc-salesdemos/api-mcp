History provides tools for retrieving resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-shape-history-get` | Get the change history for a bot detection rule. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `id` | ID x-required... | `-` |
| `namespace` | Namespace x-required... | `system` |
