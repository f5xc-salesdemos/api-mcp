Get Cloud Init Config provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-users-get-cloud-init-config-list` | Get Cloud Init Config. |

## Parameters

### Query Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `provider` | Provider for that cloud-init config. | `-` |
| `site_name` | Site name for this cloud-init config. | `-` |


## Example Usage

```bash
# List resources
```
