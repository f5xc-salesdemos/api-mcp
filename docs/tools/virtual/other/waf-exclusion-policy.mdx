WAF Exclusion Policy provides tools for creating, listing, retrieving, updating, and deleting resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-virtual-waf-exclusion-policy-create` | Create WAF Exclusion Policy. |
| `f5xc-api-virtual-waf-exclusion-policy-get` | Get WAF Exclusion Policy. |
| `f5xc-api-virtual-waf-exclusion-policy-list` | List WAF Exclusion Policy. |
| `f5xc-api-virtual-waf-exclusion-policy-update` | Update WAF Exclusion Policy. |
| `f5xc-api-virtual-waf-exclusion-policy-delete` | Delete WAF Exclusion Policy. |

## Parameters

| Parameter | Description | Example |
