Vulnerabilitie provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-virtual-vulnerabilitie-create` | Get Vulnerabilities for Virtual Host. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `name` | Virtual Host Name Virtual Host name for current request. | `my-resource` |
| `namespace` | Namespace Namespace of the virtual host for current request. | `system` |


## Configuration Choices

This resource includes mutually exclusive configuration options:

### context

| Option | Description | Recommended |
|--------|-------------|-------------|
| `api_endpoint` | Api endpoint |  |


## Example Usage

Ask Claude to help you work with Vulnerabilitie resources:

### Create Vulnerabilitie

> "Create a vulnerabilitie named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/ml/data/namespaces/$${PARAM}/virtual_hosts/$${PARAM}/vulnerabilities" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @vulnerabilitie.json

```
