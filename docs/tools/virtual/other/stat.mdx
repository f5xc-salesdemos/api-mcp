Stat provides tools for retrieving resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-virtual-stat-get` | Get API Endpoints Stats for Virtual Host. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `name` | Virtual Host Name Virtual Host for current request. | `my-resource` |
| `namespace` | Namespace Namespace of the App type for current request. | `system` |


## Example Usage

### Get Stat Details

> "Get details of the stat named 'example' in namespace 'production'"

## CURL Examples

```bash
# Get specific resource
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/ml/data/namespaces/$${PARAM}/virtual_hosts/$${PARAM}/api_endpoints/stats" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
