Sources Openapi Schema provides tools for retrieving resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-virtual-sources-openapi-schema-get` | Get relevant source OpenApi schema per API endpoint. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `name` | Virtual Host Virtual Host Name for current request. | `example-resource` |
| `namespace` | Namespace Namespace of the App type for current request. | `system` |

### Query Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `discovery_source_types` | List of wanted discovery source types... | `-` |
| `id` | Endpoint ID. | `-` |
