Load Balancer provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-virtual-loadbalancer-create` | Search load balancers All Namespaces. |

## Example Usage

Ask Claude to help you work with Load Balancer resources:

### Create Load Balancer

> "Create a loadbalancer named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/data/namespaces/system/app_security/all_ns_search/loadbalancers" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @loadbalancer.json

```
