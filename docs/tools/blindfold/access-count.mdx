Access Count provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-blindfold-access-count-create` | VoltShare Access Count Query. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace namespace is used to scope the security events for the given namespace. | `system` |


## Example Usage

Ask Claude to help you work with Access Count resources:

### Create Access Count

> "Create a access-count named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/secret_management/namespaces/$${PARAM}/voltshare/access_count" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @access_count.json

```
