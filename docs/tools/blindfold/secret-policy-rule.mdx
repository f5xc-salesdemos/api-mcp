## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/secret_management/namespaces/$${PARAM}/secret_policy_rules" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

# Get specific resource
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/secret_management/namespaces/$${PARAM}/secret_policy_rules/$${PARAM}" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \

# Delete resource

  -H "Authorization: APIToken ${F5XC_API_TOKEN}"
```
