Confirm provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-statistics-confirm-create` | Confirm Alert Receiver. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `name` | Name Name of the alert receiver. | `example-resource` |
| `namespace` | Namespace... | `system` |


## Example Usage

### Create Confirm

> "Create a confirm named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/alert/namespaces/$${PARAM}/alert_receivers/$${PARAM}/confirm" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @confirm.json

```
