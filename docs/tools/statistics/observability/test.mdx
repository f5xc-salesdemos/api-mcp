Test provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-statistics-test-create` | Test Alert Receiver. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `name` | Name Name of the alert receiver. | `my-resource` |
| `namespace` | Namespace... | `system` |


## Example Usage

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/alert/namespaces/$${PARAM}/alert_receivers/$${PARAM}/test" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @test.json

```
