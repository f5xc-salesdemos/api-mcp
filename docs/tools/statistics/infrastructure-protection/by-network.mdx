By Network provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-statistics-by-network-create` | L3l4 Network Traffic Query. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace This request is supported only in system namespace. | `system` |
| `network_id` | NetworkId The network ID. | `-` |


## Example Usage

Ask Claude to help you work with By Network resources:

### Create By Network

> "Create a by-network named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/infraprotect/namespaces/$${PARAM}/graph/l3l4/by_network/${PARAM}" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @by_network.json

```
