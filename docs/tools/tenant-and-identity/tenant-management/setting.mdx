Setting provides tools for listing, and updating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-tenantandidentity-setting-list` | GetIDMSettings. |
| `f5xc-api-tenantandidentity-setting-update` | UpdateIDMSettings. |

## Example Usage

Ask Claude to help you work with Setting resources:

### List Settings

> "List all settings in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/web/namespaces/system/tenant/idm/settings" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
