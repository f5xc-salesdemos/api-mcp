GetLoginEventsInTimeFrame returns login events for specified period of time. It consider all users
specified by context tenant.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-tenantandidentity-login-in-time-create` | GetLoginEventsInTimeFrame. |
