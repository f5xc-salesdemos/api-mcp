---
title: Enable
description: Enable tenant level OTP.
---

Enable tenant level OTP enables OTP on tenant-level. It enforces each user within a tenant to enable
OTP.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-tenantandidentity-enable-update` | Enable tenant level OTP. |
