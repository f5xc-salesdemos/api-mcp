Countrie provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-tenantandidentity-countrie-list` | List countries. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `prefix` | Prefix first two or more characters of the country you look for (can be upper or lower case) | `-` |


## Example Usage

Ask Claude to help you work with Countrie resources:

### List Countries

> "List all countries in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/web/custom/countries/${PARAM}" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
