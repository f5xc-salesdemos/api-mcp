Whoami provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-tenantandidentity-whoami-list` | Get User Details. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Namespace of the user object (namespace where the user object is stored). | `system` |


## Example Usage

Ask Claude to help you work with Whoami resources:

### List Whoamis

> "List all whoamis in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/web/custom/namespaces/$${PARAM}/whoami" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
