User Role provides tools for creating, listing, and updating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-tenantandidentity-user-role-create` | Create User with Role Assignment. |
| `f5xc-api-tenantandidentity-user-role-list` | Get User and Role Assignments. |
| `f5xc-api-tenantandidentity-user-role-update` | Update User and Role Assignments. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace All users of a tenant must be created in tenant's system namespace. | `system` |


## Example Usage

Ask Claude to help you work with User Role resources:

### Create User Role

> "Create a user-role named 'example' in the 'production' namespace"

### List User Roles

> "List all user-roles in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/web/custom/namespaces/$${PARAM}/user_roles" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/web/custom/namespaces/$${PARAM}/user_roles" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @user_role.json

```
