Usage provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-containerservices-usage-create` | Usage Metrics. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace namespace is used to scope the workload usage to a given namespace. | `system` |

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/data/namespaces/$${PARAM}/workloads/usage" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @usage.json

```
