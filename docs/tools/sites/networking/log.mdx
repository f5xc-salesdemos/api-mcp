Log provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-sites-log-create` | External connector log query. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `external_connector` | External connector External connector for which logs has to be fetched. | `-` |
| `namespace` | Namespace GET aggregation data for a given namespace. | `system` |
| `site` | Site Site where the K8s Cluster is running. | `-` |


## Example Usage

Ask Claude to help you work with Log resources:

### Create Log

> "Create a log named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/data/namespaces/$${PARAM}/site/${PARAM}/external_connector/${PARAM}/logs" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @log.json

```
