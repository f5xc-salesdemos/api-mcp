Endpoint provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-sites-endpoint-list` | Endpoints List. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `site` | Site Site name. | `-` |

### Query Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace to scope the listing of endpoints in a site. | `system` |


## Example Usage

Ask Claude to help you work with Endpoint resources:

### List Endpoints

> "List all endpoints in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/data/namespaces/system/site/${PARAM}/api/v1/endpoints" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
