Pod provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-sites-pod-list` | Pod List. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Namespace to scope the listing of pods in a site. | `system` |
| `site` | Site Site name. | `-` |


## Example Usage

Ask Claude to help you work with Pod resources:

### List Pods

> "List all pods in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/data/namespaces/system/site/${PARAM}/api/v1/namespaces/$${PARAM}/pods" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
