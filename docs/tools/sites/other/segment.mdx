Segment provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-sites-segment-list` | Segment List. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Site namespace. | `system` |
| `site` | Site... | `-` |


## Example Usage

Ask Claude to help you work with Segment resources:

### List Segments

> "List all segments in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/config/namespaces/$${PARAM}/sites/${PARAM}/segments" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
