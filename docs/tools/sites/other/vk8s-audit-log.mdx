## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace fetch vK8s audit logs for a given namespace. | `system` |

```bash
# Create resource

  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @vk8s_audit_log.json

```
