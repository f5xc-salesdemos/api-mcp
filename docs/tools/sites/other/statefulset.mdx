Statefulset provides tools for listing resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-sites-statefulset-list` | StatefulSet List. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `namespace` | Namespace Namespace to scope the listing of stateful sets in a site. | `system` |
| `site` | Site Site name. | `-` |


## Example Usage

Ask Claude to help you work with Statefulset resources:

### List Statefulsets

> "List all statefulsets in the 'production' namespace"

## CURL Examples

```bash
# List resources
curl -X GET "https://${TENANT}.console.ves.volterra.io/api/data/namespaces/system/site/${PARAM}/apis/apps/v1/namespaces/$${PARAM}/statefulsets" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}"

```
