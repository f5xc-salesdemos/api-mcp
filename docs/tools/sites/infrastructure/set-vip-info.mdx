Set VIP Info provides tools for creating resources in F5 Distributed Cloud.

## Tools

| Tool | Description |
|------|-------------|
| `f5xc-api-sites-set-vip-info-create` | Configure AWS TGW Site VIP Information. |

## Parameters

### Path Parameters

| Parameter | Description | Example |
|-----------|-------------|--------|
| `name` | Name Name of the object to be configured. | `example-resource` |
| `namespace` | Namespace Namespace for the object to be configured. | `system` |


## Example Usage

Ask Claude to help you work with Set VIP Info resources:

### Create Set VIP Info

> "Create a set-vip-info named 'example' in the 'production' namespace"

## CURL Examples

```bash
# Create resource
curl -X POST "https://${TENANT}.console.ves.volterra.io/api/config/namespaces/$${PARAM}/aws_tgw_site/$${PARAM}/set_vip_info" \
  -H "Authorization: APIToken ${F5XC_API_TOKEN}" \
  -H "Content-Type: application/json" \
  -d @set_vip_info.json

```
